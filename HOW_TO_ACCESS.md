# iPhoneからアクセスする方法

iPhoneでGohanLogを使うには、MacとiPhoneが同じWi-Fiネットワークに接続されている必要があります。

## 手順

1. **Macでサーバーを起動する**
   ターミナルで以下のコマンドを実行してください。
   ```bash
   npm run dev:host
   ```
   ※ `dev` ではなく `dev:host` を使うのがポイントです。

   **【重要】オフライン動作を確認する場合**
   開発モード（dev）ではオフライン動作が不安定な場合があります。
   確実にオフライン動作を確認するには、以下のコマンドで「本番ビルド」を実行してください。
   ```bash
   npm run build
   npm run preview:host
   ```

2. **IPアドレスを確認する**
   コマンドを実行すると、ターミナルに以下のように表示されます。
   ```
     ➜  Local:   http://localhost:5173/
     ➜  Network: http://192.168.1.5:5173/  <-- これ！
   ```
   この `Network` の横にあるURL（例: `http://192.168.1.5:5173/`）を使います。

3. **iPhoneでアクセスする**
   iPhoneのSafariを開き、上記のURLを入力してアクセスしてください。

4. **ホーム画面に追加する**
   画面が表示されたら、Safariの下部メニューにある「共有アイコン（四角から矢印）」をタップし、「ホーム画面に追加」を選択してください。
   これでアプリとして使えるようになります。
